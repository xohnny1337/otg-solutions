---
name: Writing style for UI text
description: No dashes (em/en), concise natural Norwegian, avoid AI-sounding patterns
type: feedback
---

No dashes in UI text. Keep copy short, natural, and conversational Norwegian. Avoid patterns that feel AI-generated (overly structured lists, filler phrases, long compound sentences with dashes).

**Why:** Dashes and verbose copy make the site feel AI-written. User wants it to sound like a real person wrote it.

**How to apply:** When writing or editing any user-facing text on this site, keep it tight. Short sentences. No em dashes or en dashes. Write like you'd actually talk.
